Welcome to the Movie API Documentation

This API provides access to a collection of movie data, including information about movies, actors, directors, and more.

MyFlix API v1.0.0

The MyFlix API is built with the REST architecture in mind. This API has self-explanatory URLs, returns JSON responses, and uses standard HTTP response codes.

Errors are returned as standard HTTP response codes. Messages detailing the error are included in the response body.


Supported Requests

Request URL HTTP Method Authorization Request body Response body
HTTP Method Endpoint Description
POST /users Create a new user
PUT /users/:id Update an existing user
PUT /movies/{id} Update the description of a specific movie
  • id (required): The ID of the movie to update.
  • description (required): The new description of the movie.
PUT /directors/{directorName} Update the bio of a director associated with multiple movies
  • directorName (required): The name of the director whose bio needs updating.
  • bio (required): The new biography of the director.
POST /users/:id/:movieTitle Add a movie to a user's favorite list
DELETE /users/:id/:movieTitle Remove a movie from a user's favorite list
DELETE /users/:id Delete a user
GET /movies Get all movies
GET /movies/:title Get a specific movie by title
GET /movies/genre/:genreName Get movies by genre
GET /movies/director/:directorName Get director information by name
GET /movies Get all movies filtered by genre and director
  • genre (required): The genre of the movies to filter by.
  • director (required): The director of the movies to filter by.
